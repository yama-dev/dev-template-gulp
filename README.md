# DEV TEMPLATE GULP

## 概要

webサイト制作用の開発環境

## How to use

- Nodejs
  - 公式 -> [nodejs](https://nodejs.org/ja/)
  - バージョン管理する場合は、Voltaの利用推奨 [volta](https://volta.sh/)

- Git

- コマンド実行

    ``` bash
    npm start
    ```
