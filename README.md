# DEV TEMPLATE GULP

## 概要

webサイト制作用の開発環境になります。

- HTML,Sass,javascriptファイルが同梱されています。

<br>

## 利用環境の構築

__1. Nodejs インストール__

  公式 -> [nodejs](https://nodejs.org/ja/)

<br>

## 使い方

### 1. dev-template-gulp をダウンロードして、作業フォルダに配置

__ダウンロード__
  -> https://github.com/yama-dev/dev-template-gulp/releases

### 2. ターミナルから、開発環境を起動

**ターミナルで以下のコマンドを実行**

``` bash
npm start
```
